# 出海潜力客户评估系统

专为高力集团销售团队打造的企业出海评估工具，帮助销售团队精准识别高质量潜在客户，判断客户所处阶段，挖掘交叉销售机会。

## 快速开始

在 Claude Code 中，当用户提到以下内容时，此技能会自动触发：
- 出海客户评估、潜在客户分析
- 出海打分、企业出海准备度
- 客户评级、出海阶段判断
- 海外布局评估

## 核心功能

1. **双轨评估** — 同时评估目标公司及其母公司
2. **动态权重** — 根据企业类型自动调整评估维度权重
3. **全行业覆盖** — 制造业、科技、消费品、服务业、金融等
4. **数据溯源** — 明确标注评分依据和数据缺口
5. **报告生成** — 输出结构化评估报告
6. **行动建议** — 生成0-6个月可执行的销售动作清单
7. **决策人识别** — 推荐关键决策人及接触话题

## 安装

将此技能添加到你的 Claude Code：
1. 下载 `SKILL.md`
2. 放入 `~/.claude/skills/overseas-evaluation/SKILL.md`

## 使用示例

```
/overseas-evaluation
分析某公司的出海前景
```

## 评分标准

| 得分区间 | 评级 | 说明 |
|---------|-----|------|
| 80-100 | A级 | 优质潜在客户，优先跟进 |
| 60-79 | B级 | 良好潜在客户，积极跟进 |
| 40-59 | C级 | 一般潜在客户，持续观察 |
| 0-39 | D级 | 暂不跟进，关注变化 |
