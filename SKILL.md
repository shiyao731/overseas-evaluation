# 出海潜力客户评估系统

专为高力集团打造的企业出海评估工具，帮助销售团队精准识别高质量潜在客户，判断客户所处阶段，挖掘交叉销售机会。

## 核心能力

1. **双轨评估** — 同时评估目标公司及其母公司，挖掘集团层面机会
2. **动态权重** — 根据企业类型自动调整评估维度权重
3. **全行业覆盖** — 制造业、科技、消费品、服务业、金融等所有行业
4. **数据溯源** — 明确标注评分依据和数据缺口，评估可信度
5. **报告生成** — 输出结构化评估报告（Markdown格式，可转换为PDF/Word）
6. **行动建议** — 生成0-6个月可执行的销售动作清单
7. **决策人识别** — 推荐关键决策人及接触话题

## 评估维度体系

### 十大核心维度（100分制）

| 维度 | 说明 | 权重范围 |
|-----|------|---------|
| 市场准入 | 目标市场进入壁垒、政策环境、准入门槛 | 5-20% |
| 基础合规 | 法律法规、资质许可、监管要求 | 5-20% |
| 风险评估 | 政治/经济/汇率/合规风险 | 5-15% |
| 运营效率 | 现有海外项目运营质量（如有） | 0-20% |
| 本地化团队 | 本地团队能力、文化适配 | 5-15% |
| 供应链整合 | 物流、采购、本地供应商 | 5-15% |
| 成本控制 | 成本结构、规模效应 | 5-20% |
| 战略协同性 | 与母公司整体战略匹配度 | 5-15% |
| 品牌影响力 | 品牌在目标市场认知度 | 5-15% |
| 技术适配性 | 产品/服务本地化程度 | 5-15% |

## 企业分类与动态权重

### 分类维度
- **出海阶段**: 初次探索 / 积极筹备 / 已有海外落地 / 成熟扩张
- **企业体量**: 大型集团(年营收>50亿) / 中型企业(5-50亿) / 小型企业(<5亿)
- **行业属性**: 制造业、科技互联网、消费品、服务业、金融、能源等

## 评分标准

| 得分区间 | 评级 | 说明 |
|---------|-----|------|
| 80-100 | A级 | 优质潜在客户，优先跟进 |
| 60-79 | B级 | 良好潜在客户，积极跟进 |
| 40-59 | C级 | 一般潜在客户，持续观察 |
| 0-39 | D级 | 暂不跟进，关注变化 |

## 使用场景

- **销售线索评估**: 打分判断是否值得跟进
- **客户需求洞察**: 识别客户核心痛点
- **交叉销售**: 挖掘集团内部其他业务机会
- **竞品分析**: 了解客户竞品布局
- **行动规划**: 制定0-6个月销售计划
- **内部汇报**: 生成客户评级报告

## 注意事项

1. **双轨评估**: 始终评估目标公司+母公司
2. **数据溯源**: 每项分数必须标注依据
3. **缺口标注**: 无法获取的数据必须明确标注
4. **动态权重**: 根据企业类型自动调整
5. **全行业**: 使用通用框架，各行业特殊需求可在维度下扩展
6. **报告输出**: 始终使用上述报告模板格式
