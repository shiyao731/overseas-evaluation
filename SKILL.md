# 出海潜力客户评估系统 v6.0

帮助销售团队精准识别高质量潜在客户，判断客户所处阶段，挖掘交叉销售机会的企业出海评估工具。

## 核心设计原则

1. **双轨评估机制** — 目标公司+母公司独立评估，挖掘集团层面机会
2. **三维动态权重** — 根据"出海阶段 + 企业体量 + 行业属性"自动计算权重
3. **微弱信号探测** — 主动识别高价值的动态预警信号
4. **置信度透明** — 每次评估附带数据完整度评分
5. **竞品FOMO触发** — 挖掘竞争对手动作作为销售突破口
6. **集团关联挖掘** — 识别母公司其他出海活跃业务板块

## 第一步：三维修正识别

### 维度1：出海阶段

| 阶段 | 定义 | 核心问题 |
|-----|------|---------|
| **阶段一：出海初期** | 刚开始考虑出海，无实质动作 | "该不该出海？从哪里开始？" |
| **阶段二：出海成长期** | 已有1-2个海外项目，运营<3年 | "如何优化现有海外业务？" |
| **阶段三：出海成熟期** | 海外业务体系化，运营>3年 | "如何加速扩张？" |

### 维度2：企业体量

| 体量 | 判定标准 | 核心关注 |
|-----|---------|---------|
| **大型集团** | 年营收>50亿，或母公司年营收>100亿 | 战略布局、风险控制 |
| **中型企业** | 年营收5-50亿 | 增长与成本平衡 |
| **小型创新企业** | 年营收<5亿 | 市场增长、技术突破 |

### 维度3：行业属性

| 行业 | 特性 | 权重侧重 |
|-----|------|---------|
| 制造业 | 供应链本地化要求高 | 供应链×1.3 |
| 科技/互联网 | 技术适配、品牌重要 | 技术×1.3、品牌×1.2 |
| 消费品 | 品牌、渠道、本地化 | 品牌×1.3 |
| 服务业 | 人员本地化、合规重要 | 本地化×1.3 |
| 金融 | 合规、风控严 | 合规×1.4 |

## 第二步：双轨评估机制

### 核心逻辑

对于每个评估对象，模型**必须同时评估两条轨道**：

```
┌─────────────────────────────────────────────────────┐
│                   双轨评估                          │
├─────────────────────┬───────────────────────────────┤
│    轨道A：目标公司    │      轨道B：母公司            │
│  (目标公司本体评估)   │   (母公司集团层面评估)        │
├─────────────────────┼───────────────────────────────┤
│ • 独立评估公司本体    │ • 评估母公司出海战略          │
│ • 判断独立出海价值    │ • 识别其他出海业务板块        │
│ • 若信息不足，基于   │ • 判断资金支持力度            │
│   业务特性评估       │ • 识别交叉销售机会            │
└─────────────────────┴───────────────────────────────┘
```

### 何时使用双轨评估

| 情况 | 处理方式 |
|-----|---------|
| 目标公司有明确母公司 | 必须双轨评估 |
| 目标公司无母公司/独立公司 | 仅做轨道A，在报告中注明 |
| 目标公司信息极度匮乏 | 侧重轨道B，以母公司信息为主 |

### 轨道A：目标公司评估

对目标公司本体进行独立评估：
1. 基于目标公司的业务特性、行业、规模
2. 如信息不足，基于业务特性推断
3. 独立给出目标公司评分

### 轨道B：母公司评估

对母公司进行集团层面评估：
1. 评估母公司的出海战略成熟度
2. **必须识别并列出**母公司旗下其他出海活跃的业务板块
3. 评估母公司对目标公司的资金/资源支持力度
4. 识别集团内部的交叉销售机会

## 注意事项

1. **双轨评估必须执行**: 每个评估都要分别给出目标公司和母公司的评分
2. **集团关联必须挖掘**: 母公司的其他出海业务板块必须识别并列出
3. **三维权重必须计算**: 根据出海阶段+体量+行业自动计算
4. **置信度必须标注**: 每个轨道单独标注置信度
5. **交叉销售必须识别**: 基于集团内部关联，识别交叉销售机会
6. **数据缺口必须透明**: 无法获取的数据必须明确标注
