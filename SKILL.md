# 出海潜力客户评估系统 v8.0

帮助销售团队精准识别高质量潜在客户，判断客户所处阶段，挖掘交叉销售机会的企业出海评估工具。

## 核心设计原则

1. **全数据溯源原则（最重要）** — 报告中所有数据必须有明确来源标注，无来源的数据不得出现在报告中
2. **双轨评估机制** — 目标公司+母公司独立评估，挖掘集团层面机会
3. **双轨竞品对标** — 竞品公司+竞品母公司双轨扫描，追踪最新动态
4. **三维动态权重** — 根据"出海阶段 + 企业体量 + 行业属性"自动计算权重
5. **微弱信号探测** — 主动识别高价值的动态预警信号
6. **置信度透明** — 每次评估附带数据完整度评分
7. **集团关联挖掘** — 识别母公司其他出海活跃业务板块

## 第一步：全数据溯源原则（必须严格执行）

### 核心要求

**报告中每一个数据点都必须标注来源**。无来源的数据不得出现在报告中。

### 数据来源分级

| 等级 | 来源类型 | 置信度 | 示例 |
|-----|---------|-------|-----|
| **A级** | 官方权威来源 | 最高 | 财报、招股书、政府文件、法院判决 |
| **B级** | 机构公开发布 | 高 | 行业协会报告、券商研报、新闻报道 |
| **C级** | 平台数据 | 中 | 招聘平台、征信平台、企业信息平台 |
| **D级** | 公开可查信息 | 中低 | 社交媒体、公开论坛、搜索引擎 |
| **E级** | 专业推断 | 低 | 基于公开信息的逻辑推断（需标注"推断"） |

### 必须标注来源的数据类型

| 数据类别 | 必须标注内容 | 示例 |
|---------|------------|-----|
| **企业基本信息** | 数据来源 | 年报、官网、企业信用信息平台 |
| **财务数据** | 具体财报来源、报表期 | "来源：2024年年报第X页" |
| **人员规模** | 来源+时间 | "来源：领英，2026年3月" |
| **海外布局** | 项目名称、签约时间、来源 | "来源：企业公告，2025年6月" |
| **竞品动态** | 具体新闻来源+时间 | "来源：Reuters，2026-01-15" |
| **行业数据** | 报告名称+发布机构 | "来源：中国汽车工业协会2025年度报告" |
| **政策信息** | 文件名称+发布机构 | "来源：商务部《对外投资合作国别指引》" |

### 禁止出现的内容

❌ "数据显示..."
❌ "据悉..."
❌ "有消息称..."
❌ "知情人士透露..."
❌ 任何无来源的数据陈述

### 正确示例

✅ "延锋年营收约200亿元（来源：企业官网公司简介，更新时间2025年）"
✅ "2024年海外销量40万辆（来源：上汽集团2024年年报，第X页）"
✅ "员工规模约2000人（来源：领英企业主页，2026年3月采集）"

## 第二步：三维修正识别

### 维度1：出海阶段

| 阶段 | 定义 | 核心问题 | 数据来源示例 |
|-----|------|---------|------------|
| **阶段一：出海初期** | 刚开始考虑出海，无实质动作 | "该不该出海？从哪里开始？" | 企业公告、招聘平台、企业官网 |
| **阶段二：出海成长期** | 已有1-2个海外项目，运营<3年 | "如何优化现有海外业务？" | 企业年报、新闻报道 |
| **阶段三：出海成熟期** | 海外业务体系化，运营>3年 | "如何加速扩张？" | 企业年报、公告 |

### 维度2：企业体量

| 体量 | 判定标准 | 核心关注 | 数据来源示例 |
|-----|---------|---------|------------|
| **大型集团** | 年营收>50亿，或母公司年营收>100亿 | 战略布局、风险控制 | 上市公司年报、招股书 |
| **中型企业** | 年营收5-50亿 | 增长与成本平衡 | 企业财报、行业协会数据 |
| **小型创新企业** | 年营收<5亿 | 市场增长、技术突破 | 企业官网、融资披露 |

### 维度3：行业属性

| 行业 | 特性 | 权重侧重 | 数据来源示例 |
|-----|------|---------|------------|
| 制造业 | 供应链本地化要求高 | 供应链×1.3 | 企业官网、行业分类目录 |
| 科技/互联网 | 技术适配，品牌重要 | 技术×1.3，品牌×1.2 | 企业官网、产品介绍 |
| 消费品 | 品牌、渠道、本地化 | 品牌×1.3 | 行业分析报告 |
| 服务业 | 人员本地化、合规重要 | 本地化×1.3 | 企业官网、行业分类 |
| 金融 | 合规、风控严 | 合规×1.4 | 监管机构文件、企业公告 |

## 第三步：双轨竞品对标（新增重点模块）

### 3.1 竞品对标核心原则

竞品分析必须遵循**双轨扫描**原则，且所有情报必须标注来源：

```
┌─────────────────────────────────────────────────────────┐
│                  双轨竞品对标                            │
├─────────────────────────┬─────────────────────────────────┤
│    竞品公司层            │      竞品母公司层               │
│  • 竞品公司本体分析       │   • 竞品母公司整体战略           │
│  • 竞品海外落地详情       │   • 母公司旗下其他出海业务       │
│  • 竞品市场策略          │   • 母公司的资源支持力度         │
├─────────────────────────┼─────────────────────────────────┤
│  必须标注来源:            │   必须标注来源:                 │
│  • 新闻报道/公告+时间     │   • 集团年报/公告               │
│  • 招聘平台数据+时间      │   • 集团战略发布会               │
│  • 行业研究报告           │   • 上市公司公告                 │
│  • 竞品官网/社媒          │   • 行业研究机构分析              │
└─────────────────────────┴─────────────────────────────────┘
```

### 3.2 竞品识别标准

**一级竞品**（同体量、同市场）：
- 在相同目标市场直接竞争
- 同体量级企业
- 出海阶段相近

**二级竞品**（同行业、不同体量/市场）：
- 行业相同，但市场或体量不同
- 可借鉴出海策略

**三级竞品**（母公司层面）：
- 母公司旗下其他业务板块已在目标市场布局
- 可作为集团出海参考

### 3.3 竞品情报收集要素

每个竞品必须收集以下情报：

#### 竞品公司层情报

| 情报要素 | 说明 | 数据来源 | 采集优先级 |
|---------|-----|---------|-----------|
| **最新动态** | 近3-6个月的出海新闻、投资、签约等 | 新闻、公告 | 必须 |
| **海外落地详情** | 已落地市场、项目规模、运营时间 | 官网、新闻、年报 | 必须 |
| **市场策略** | 目标市场选择、定位、渠道策略 | 行业分析、官网 | 必须 |
| **组织架构** | 海外团队规模、组织架构 | 领英、招聘平台 | 应收集 |
| **财务表现** | 海外营收、投入产出比 | 财报、分析报告 | 应收集 |
| **合作伙伴** | 当地合作伙伴、经销商 | 新闻、官网 | 可选 |

#### 竞品母公司层情报

| 情报要素 | 说明 | 数据来源 | 采集优先级 |
|---------|-----|---------|-----------|
| **集团出海战略** | 整体国际化战略、目标、规划 | 年报、战略发布会 | 必须 |
| **其他出海业务板块** | 集团旗下其他已出海的业务单元 | 年报、官网 | 必须 |
| **资源投入** | 对该竞品的资源倾斜程度 | 公告、投资计划 | 应收集 |
| **成功/失败案例** | 集团出海的成功失败经验 | 行业分析、新闻 | 应收集 |

### 3.4 竞品动态追踪时间线

| 时间范围 | 情报类型 | 价值说明 |
|---------|---------|---------|
| 近1个月 | 突发动态 | 重大投资、战略调整 |
| 1-3个月 | 近期动态 | 招聘、新签合同、市场进入 |
| 3-6个月 | 中期动态 | 项目落地、运营进展 |
| 6-12个月 | 战略动向 | 战略布局调整、资源重新配置 |

## 注意事项

1. **全数据溯源必须执行（最重要）**: 报告中每个数据都必须标注来源，无来源=不可使用
2. **禁止无来源陈述**: 不得出现"数据显示"、"据悉"等无来源表述
3. **双轨竞品对标必须执行**: 每个竞品必须同时扫描竞品公司层和竞品母公司层
4. **竞品情报时效性**: 重点关注近3-6个月的最新动态，标注采集时间
5. **竞品分级管理**: 一级竞品必须深度分析，二级竞品适度关注，三级竞品供参考
6. **双轨评估仍需执行**: 目标公司+母公司独立评估
7. **三维权重必须计算**: 根据出海阶段+体量+行业自动计算
8. **置信度必须标注**: 每个轨道单独标注置信度
